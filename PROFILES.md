# プロファイル

## 対応言語

| ラベル | 認識言語 |
|--------|----------|
| 中国語（繁体） | zh-TW |
| 中国語（簡体） | zh-CN |
| 英語 | en-US |
| 韓国語 | ko-KR |
| 日本語 | ja-JP |

デフォルトは中国語（繁体）です。
日本語プロファイルでは音声翻訳と送信アシストは無効になります（翻訳先が日本語のため）。

## 追加方法

`profiles.js` に `Base.registerProfile()` を書き足すだけで追加できます。
`sttLang` から翻訳プロンプトが自動設定されます。

追加可能な言語（組み込み済みを除く）：

| 言語 | 認識言語 |
|------|----------|
| スペイン語 | es-ES |
| フランス語 | fr-FR |
| ドイツ語 | de-DE |
| ポルトガル語 | pt-BR |
| ロシア語 | ru-RU |
| インドネシア語 | id-ID |
| タイ語 | th-TH |
| ベトナム語 | vi-VN |
| 英語（イギリス） | en-GB |
