# Ubye JP Assist

台湾 VTuber「悠白（Ubye）」の配信を日本語で理解し、リアルタイムで楽しむために作った Chrome 拡張機能です。
字幕・チャット・コメントを翻訳し、日本語リスナーの視聴体験をサポートします。
言語プロファイルの切り替えで、他の言語の配信にも対応できます。

## 主な機能

- **リアルタイム字幕** — タブ音声を認識し、字幕として表示。自動翻訳にも対応（任意のサイトで利用可能）
- **チャット翻訳** — ライブチャット・投票をバッチ翻訳。自動翻訳タイマー付き（YouTube）
- **コメント翻訳** — アーカイブのコメント欄を一括翻訳（YouTube）
- **送信アシスト** — 日本語入力を配信者の言語に変換して入力欄にセット（YouTube）
- **言語プロファイル** — 音声認識言語・翻訳・送信アシストをプロファイルで一括切り替え
- **表示設定** — フォントサイズ・字幕フェード時間などをポップアップから調整

> YouTube 以外のサイトではリアルタイム字幕のみ利用できます。ポップアップの開始ボタンから起動してください。

## 翻訳エンジン

機能ごとに翻訳エンジンを選択できます（初期設定は無料翻訳）。

| エンジン | 特徴 |
|----------|------|
| **無料翻訳（内蔵）** | Chrome 内蔵の Translator API |
| **Gemini Flash Lite** | Google の Gemini API（API Key 必要） |

- より自然な翻訳には Gemini モデルを推奨します
- 送信アシストは投稿内容に直結するため Gemini 専用です

## 動作要件

- Google Chrome 138 以降
- Gemini API キー（Gemini モデルを使う場合。[Google AI Studio](https://aistudio.google.com/) で取得）

## セットアップ

1. `chrome://extensions` →「デベロッパーモード」→「パッケージ化されていない拡張機能を読み込む」でこのフォルダを選択
2. 言語プロファイルを選択し、各機能の ON/OFF とモデルを設定
3. Gemini モデルを使う場合は、[Google AI Studio](https://aistudio.google.com/) で API キーを取得し、翻訳設定に入力して保存

> Gemini API は従量課金制です。本拡張では低コストな Lite モデルを採用しています。料金は [Gemini API Pricing](https://ai.google.dev/pricing) を確認してください。

> **API キーの取り扱い:** API キーは端末内に保存され、外部サービスへは送信されません。翻訳リクエスト時にのみ Google Gemini API へ送信されます。

## 制限事項

- YouTube の仕様変更により動作しなくなる可能性があります
- Web Speech API の認識精度はブラウザや環境に依存します
- 翻訳の品質・速度は選択したエンジンに依存します
- 音声認識はタブごとに1つまでです（Chrome の制約）
